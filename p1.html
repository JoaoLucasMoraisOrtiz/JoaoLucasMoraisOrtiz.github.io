Resposta das perguntas

Quais as funções da tag <meta>? Dê exemplo.

As tags meta definem os meta dados do site.
Entre os principais metadados estão:
charset -> define o tipo de caracteres utilizados no site, no portugues, o UTF-8;
name -> define um nome para o meta dado;
content -> especifíca o valor a ser atribuido ao metadado com o nome definido na tag name;
description -> cria uma descrição da página;
viewport -> determina as dimensões da janela que seu site será exibido -> possui width e reight para definir sua largura e altura, e initial-scale para definir a escala inicial de visualização da página;
keywords -> aqui você declara as palavras chaves do site que vão ser utilizadas por buscadores para localizar seu site;
